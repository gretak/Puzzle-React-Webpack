# Webpack-react
segment with react, using webpack


--------------
1. need to have node.js installed
2. install webpack:
 $ npm install webpack -g
3. create entry.js and index.html files (full example onhttps://webpack.github.io/docs/tutorials/getting-started/)
4. run:
 $ webpack ./entry.js bundle.js


---------------
5. npm install css-loader style-loader to install the loaders.

---------------
run webpack
